# 🚀 Manpower Station Technologies

A scalable and production-ready **MERN Stack** web application designed to streamline manpower recruitment and workforce management. The platform connects employers with job seekers, enabling efficient job postings, candidate applications, and administrative control through a secure and user-friendly interface.

---


## 📖 Overview

**Manpower Station Technologies** is a full-stack recruitment and workforce management system that simplifies the hiring process. Employers can post job opportunities, manage applicants, and track recruitment progress, while candidates can search for jobs and submit applications seamlessly.

The application follows modern software engineering practices, including RESTful API design, JWT-based authentication, scalable architecture, and containerized deployment.

## ✨ Features

### 👤 User Management
- User registration and login
- JWT-based authentication
- Role-based access control (Admin, Employer, Candidate)
- Profile management

### 💼 Job Management
- Create, update, and delete job postings
- Advanced job search and filtering
- Pagination and sorting

### 📄 Application Management
- Candidates can apply for jobs
- Employers can review and manage applications
- Resume/CV upload functionality

### 📊 Admin Dashboard
- Manage users, jobs, and applications
- Analytics and reporting

### 🔔 Notifications (Optional)
- Real-time notifications using Socket.io
- Email notifications for job updates

### 📱 Responsive UI
- Mobile-friendly and accessible design
- Modern UI built with Tailwind CSS or Bootstrap
